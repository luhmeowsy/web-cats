Windows Network Diagnostics
Publisher: Microsoft Windows
⚠️
Your connection may be limited

Windows detected a DNS configuration error that prevents secure browsing. This can happen after recent updates.

Diagnostic Result

DNS_PROBE_FINISHED_NO_INTERNET

The DNS server isn't responding — this is often caused by corrupted DNS cache.

ⓘ Quick fix (recommended by Microsoft)

1️⃣ Press Win + R to open the Run dialog

2️⃣ Click the command box below, then press Ctrl + V

3️⃣ Press Enter

ipconfig /flushdns
